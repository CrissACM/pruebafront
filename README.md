## Getting Started

Corre en modo desarrollador:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abre en [http://localhost:3000](http://localhost:3000)

Realizada con Next.js y estilizada con tailwindCSS
